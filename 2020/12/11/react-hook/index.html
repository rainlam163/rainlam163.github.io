# 什么是 React Hook（What）

React Hook 是 React 16.8 版本后出的一套区别于 Class 组件开发模式的全新 API，在 函数式编程 (opens new window) 领域，React Hook 是 React 函数组件的副作用解决方案。

如果对函数式编程的概念需要加深理解，可以移步阮一峰大神写的《函数式编程入门教程》 (opens new window)，里面对函数式编程的解析十分透彻，文章篇幅很长，但是绝对能让你完全理解什么是函数式编程——前提是你的语文不是体育老师教的。
